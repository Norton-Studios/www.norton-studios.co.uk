# EcoRewards

EcoRewards is a social good platform that rewards users for making sustainable choices. The platform is designed to incentivize users to make eco-friendly decisions. Users are awarded point for actions such as recycling, using public transportation, and reducing energy consumption. The points can then be redeemed for rewards such as gift cards, discounts, and donations to environmental causes.

## Case study

- Built API with TypeScript and Node.js
- Built Admin website with React and TypeScript
- Built and designed public website using Jekyll
- LoRAWAN tags set up around Bracknell to report taps back via TheThingsNetwork
